## Key Features to Validate:

1. **Grid Layout**
   - Rows = Subjects (S01, S02, S03...)
   - Columns = Features (joint angles)
   - Each subplot is one subject-feature combination

2. **Phase Axis**
   - X-axis: 0-100% gait cycle
   - Should see smooth progression, not jumps

3. **Value Ranges** (for level walking)
   - Hip: 0.3-0.6 rad (17-34 degrees)
   - Knee: 0.95-1.2 rad (54-69 degrees)  
   - Ankle: -0.3-0.25 rad (-17-14 degrees)

4. **Patterns**
   - Sinusoidal-like patterns for joint angles
   - Consistent cycle-to-cycle in individual steps view
   - Reasonable STD bands (not too wide or narrow)

5. **Data Quality Indicators**
   - No gaps or discontinuities
   - No extreme outliers
   - Similar patterns across subjects
   - 150 points per cycle (smooth curves)
