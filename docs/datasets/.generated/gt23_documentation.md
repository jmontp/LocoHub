## Overview

- **Short Code**: GT23
- **Year**: 2023
- **Institution**: Epic Lab

Georgia Tech 2023 dataset containing comprehensive biomechanical data from able-bodied adults performing multiple locomotion tasks including level walking, incline/decline walking, stair ascent/descent, sit-to-stand, stand-to-sit, squats, jumps, lunges, cutting, side shuffle, and various agility drills. Data collected using motion capture and force plates.
